# 开发维护指南

## 目录约定

- MATLAB 源码放入 `matlab/`。
- Python 源码放入 `python/`。
- Windows 启动入口只保留 `start_project.bat`。
- 生成数据放入 `data/`、`models/`、`main/SimulationData/`、`main/Imgs/`。

## 修改参数

优先修改：

```text
matlab/config/getDefaultParams.m
```

不要把实验参数散落到算法函数中。

## 修改路径

MATLAB 路径：

```text
matlab/config/getProjectPaths.m
```

Python 路径：

```text
python/project_paths.py
```

不要在模块中写死本机绝对路径。

## 新增 MATLAB 算法

1. 将函数放到 `matlab/power_allocation/` 或对应模块。
2. 在 `matlab/main/Combined_Downlink_Sim.m` 中注册方法。
3. 补充必要的性能统计字段。
4. 运行 `start_project.bat 5 /y` 做烟测。

## 新增 Python 模型

1. 在 `python/` 中添加训练脚本或运行时。
2. 将模型输出放到 `models/`。
3. 在 MATLAB 侧添加或复用 `computeRho*.m` 适配器。
4. 在 `getDefaultParams.m` 中集中配置模型文件名。

## 验证建议

轻量检查：

```bat
start_project.bat C /y
start_project.bat 6 /y
start_project.bat 5 /y
```

完整检查：

```bat
start_project.bat 1 /y
```
