# Windows 启动与运行流程

项目只保留一个 `.bat` 启动脚本：

```bat
start_project.bat
```

双击后会出现菜单。也可以在命令行中带参数运行：

```bat
start_project.bat C /y
start_project.bat 3 /y
start_project.bat 6 /y
```

## 菜单说明

- `1` 快速完整流程：导出小数据、训练模型、烟测、主仿真。
- `2` 完整实验流程：按 `matlab/config/getDefaultParams.m` 的规模运行。
- `3` 仅主仿真：复用已有数据、模型和缓存。
- `4` 仅重绘图：从 `main/SimulationData/Simulation_Results_v2.mat` 重新生成图。
- `5` 仅烟测：检查 Python 推理和 MATLAB-Python 桥接。
- `6` 验证最新训练数据集。
- `7` 快速导出训练数据：每个 SNR 只导出 2 个快照。
- `8` 打开 MATLAB GUI 并加入 `matlab/` 路径。
- `C` 环境检查。
- `Q` 退出。

## 常用参数

- `/y`：运行结束不暂停。
- `/retrain`：已有模型也重新训练。

示例：

```bat
start_project.bat 1 /y
start_project.bat 1 /y /retrain
```

## 依赖检查

启动脚本会检查：

- `matlab` 是否在 PATH 中。
- `python` 是否在 PATH 中。
- `matlab/`、`python/`、`data/gnn_training/`、`models/` 是否存在。

如果 MATLAB 不在 PATH 中，需要把 MATLAB 的 `bin` 目录加入系统 PATH，例如：

```text
C:\Program Files\MATLAB\R2025a\bin
```
