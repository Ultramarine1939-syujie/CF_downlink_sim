# Python 训练与推理说明

Python 代码位于 `python/`。

## 依赖安装

```bat
cd python
pip install -r requirements.txt
```

## 路径管理

Python 侧统一路径文件：

```text
python/project_paths.py
```

训练脚本默认会使用：

- 数据：`data/gnn_training/*.mat`
- 输出模型：`models/`

## 数据集

`python/dataset.py` 读取 MATLAB v7.3 `.mat` 文件，并将 HDF5 中的维度整理为 `(L, K, snapshots)`。

数据由 MATLAB 函数生成：

```matlab
exportTrainingData()
```

快速导出可使用：

```bat
start_project.bat 7 /y
```

## 训练脚本

- `python/train_gnn.py`：训练 GNN 或 DCGNN。
- `python/train_gnn_unsup.py`：训练 U-GNN。
- `python/train_gnn_local.py`：训练 Local-GNN。
- `python/train_rl_power.py`：训练 DQN/DDPG。

示例：

```bat
cd python
python train_gnn.py --data "../data/gnn_training/*.mat" --epochs 100 --output_dir "../models"
python train_gnn.py --data "../data/gnn_training/*.mat" --epochs 100 --output_dir "../models" --model_type dcgnn
python train_gnn_local.py --data "../data/gnn_training/*.mat" --epochs 100 --output_dir "../models"
```

## MATLAB 调用的推理运行时

- `python/gnn_runtime.py`
- `python/gnn_runtime_local.py`
- `python/rl_runtime.py`

这些运行时会缓存模型，减少 MATLAB 循环调用时的加载开销。
