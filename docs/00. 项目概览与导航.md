## 一、项目目标

### 核心研究问题

在 **Cell-Free 分布式 MIMO** 下行链路中，如何设计**低复杂度、低同步时延**的功率分配方案，使其逼近集中式最优（WMMSE）性能？

### 技术路线

```
传统方案（高复杂度/高时延）          本项目创新方案
┌──────────────────────┐          ┌───────────────────────────────┐
│ 集中式 WMMSE         │   vs     │ GNN 本地推理 + 联邦学习协作    │
│ - CPU 收集全部 CSI    │          │ - 每个 AP 独立推理，无需通信    │
│ - 集中计算 ρ 矩阵      │   ──→    │ - FedAvg 聚合弥补信息损失      │
│ - O(L×K) 同步代价     │          │ - O(K) 通信量，10~20轮收敛     │
└──────────────────────┘          └───────────────────────────────┘
  性能最优但时延大                     推近最优且时延极低
```

## 五、关键概念速查

### 接入模式

| 模式 | 说明 | 复杂度 | 性能 |
|------|------|--------|------|
| **All** | 所有 AP 服务所有 UE（理论上界） | O(L×K) | 最高 |
| **DCC** | 每个 UE 只选部分 AP（实际系统） | 降低 | 接近 All |

### 预编码方法

| 方法 | 干扰抑制 | 信息范围 | 复杂度 |
|------|---------|---------|--------|
| **MR** | ❌ 不抑制 | 本地 AP | ⭐ 最低 |
| **R-MMSE** (LP-MMSE) | ✔ 局部抑制 | 本地 UE | ⭐⭐ 中等 |
| **L-MMSE** | ✔ 全局抑制 | 全局 UE | ❌❌❌ 最高 |

### 功率分配方法

| 方法 | 特点 | 角色 |
|------|------|------|
| **Baseline (Dist)** | 基于大尺度衰落比例 | 基线对比 |
| **EPA** | 等功率分配 | 简单下界 |
| **Random** | 随机生成后归一化 | 对照组 |
| **PSO** | 粒子群优化最大化 ESR | 中等性能 |
| **WMMSE** | 加权最小均方误差迭代 | **性能上界 + GNN训练标签** |
| **GNN** (新) | 图神经网络推理 | **替代 WMMSE，低延迟部署** |
