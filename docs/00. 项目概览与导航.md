# 项目概览与导航

本项目用于研究 Cell-Free Massive MIMO 下行链路中的预编码、功率分配、同步开销和学习型功率控制方法。

当前代码结构：

- `start_project.bat`：唯一 Windows 启动脚本。
- `run.m`：MATLAB 主入口。
- `matlab/`：MATLAB 仿真代码。
- `python/`：Python 训练和推理代码。
- `data/gnn_training/`：训练数据。
- `models/`：模型权重。
- `main/SimulationData/`：仿真缓存和结果。
- `main/Imgs/`：输出图像。
