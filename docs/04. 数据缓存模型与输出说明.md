# 数据、缓存、模型与输出说明

## 训练数据

目录：

```text
data/gnn_training/
```

文件名格式：

```text
gnn_training_data_YYYYMMDD_HHMMSS.mat
```

生成方式：

```bat
start_project.bat 7 /y
```

或在 MATLAB 中：

```matlab
exportTrainingData()
```

## 模型

目录：

```text
models/
```

常用模型文件：

- `best_gat_gnn_power.pt`
- `best_dcgnn_power.pt`
- `best_local_gnn_power.pt`
- `best_ugnn_power.pt`
- `best_dqn_power.pt`
- `best_ddpg_power.pt`

如果某个模型缺失，主仿真中的对应学习方法会回退到 EPA，避免整个仿真中断。

## 仿真缓存与结果

目录：

```text
main/SimulationData/
```

典型文件：

- `cache_scenario_s*.mat`
- `cache_snr_s*_sn*.mat`
- `Simulation_Results_v2.mat`
- `Sync_Ablation_Results.mat`
- `Sync_Ablation_Table.csv`

缓存由 `params.runtime.useCache` 控制。

## 图像

目录：

```text
main/Imgs/
```

主仿真会输出 ESR 曲线、方法对比图、时间开销图和同步时延消融图。

重绘图：

```bat
start_project.bat 4 /y
```
