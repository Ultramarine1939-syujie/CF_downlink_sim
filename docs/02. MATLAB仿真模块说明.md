# MATLAB 仿真模块说明

MATLAB 代码位于 `matlab/`。

## 入口

根目录 `run.m` 是主入口。它会加入 `matlab/` 路径，然后进入 `matlab/main/` 执行 `Combined_Downlink_Sim`。

## 配置

核心配置文件：

```text
matlab/config/getDefaultParams.m
```

路径配置文件：

```text
matlab/config/getProjectPaths.m
```

修改实验规模、SNR、缓存、模型文件名时，优先修改 `getDefaultParams.m`。

## 主仿真

`matlab/main/Combined_Downlink_Sim.m` 负责：

1. 读取参数和路径。
2. 注册功率分配方法和预编码方法。
3. 生成或读取场景缓存。
4. 按 SNR 扫描。
5. 计算传统功率分配方法。
6. 调用 Python 模型推理学习型方法。
7. 计算频谱效率和 ESR。
8. 保存缓存、结果和图像。

## 主要模块

- `matlab/channel/`：场景、DCC 连接、信道估计。
- `matlab/precoding/`：MR、L-MMSE、L-MMSE-G、R-MMSE。
- `matlab/power_allocation/`：EPA、FPCP、D-WMMSE、WMMSE、GNN、Local-GNN、U-GNN、DQN、DDPG。
- `matlab/se_calculation/`：SE 计算。
- `matlab/visualization/`：结果打印和绘图。

## 与 Python 的连接

MATLAB 通过以下函数调用 Python 运行时：

- `computeRhoGNN.m`
- `computeRhoLocalGNN.m`
- `computeRhoRL.m`

这些函数会把 `python/` 加入 Python 路径，并调用对应的运行时模块。
